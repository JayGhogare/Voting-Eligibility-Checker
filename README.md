# 🗳️ Voting Eligibility Checker

A simple Python project that checks whether a person is eligible to vote based on their age.

## 📌 Features
- Takes username input
- Takes age/password input
- Checks eligibility using condition (if-else)
- Beginner-friendly logic project

## 💻 Tech Used
- Python

## 🧠 Logic
If age is 18 or above → Eligible to vote  
If age is below 18 → Not eligible to vote

## 🚀 How to Run
1. Open project in VS Code
2. Run Python file
3. Enter username and age when asked
4. Get eligibility result
